# 隐私与安全说明

本项目默认本地运行，不需要账号、密钥或云端服务。

## 数据来源

只处理用户显式提供的文件或文本，例如：

- Codex `/status` 文本。
- 手工 JSON 用量样本。
- 用户选择的 Skill、提示词或输出文本。

## 不做的事

- 不登录 OpenAI、ChatGPT 或 Codex。
- 不读取浏览器 cookie、系统凭据或钥匙串。
- 不代理、拦截或修改 Codex 请求。
- 不绕过登录、手机号验证、订阅、地区或官方计费。
- 不上传报告、原文或统计到云端。
- 不自动安装、启用或修改 Skill。

## 脱敏

报告会尝试脱敏：

- OpenAI 风格 API key。
- `api_key`、`token`、`secret`、`access_token` 字段。
- Cookie 和 Bearer token。
- 邮箱和中国大陆手机号。

脱敏不是安全边界的唯一来源。用户仍应避免把真实密钥放入输入文件。

## 删除

执行：

```bash
probe --db .probe/demo.db delete --all --yes
```

会删除本地业务数据，包括导入批次、用量记录、风险条目、报告索引和验证事件。工具会保留最小审计日志，用于证明删除动作发生。

## 风险声明

这是 Watch / 验证型工具，不是商业化 Go 产品。报告只提供人工复核建议，不保证节省额度、不保证平台过审、不替代官方 dashboard 或 `/status`。
